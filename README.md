# Interactive Supplementary Graphic

This file, `index.html`, is an interactive supplementary graphic associated with:

**Khavari et al. Ultrasound Neuromodulation for Dementia: A Scoping Review of Emerging Clinical Evidence**

## Description

The supplementary graphic is provided as a standalone HTML file and is designed for viewing in a standard web browser. It uses Plotly to display an interactive visualization that complements the findings presented in the manuscript. :contentReference[oaicite:2]{index=2}

## Viewing Instructions

To view the supplementary graphic:

1. Download or save `index.html`
2. Open the file in a modern web browser

Compatible browsers include Chrome, Safari, Firefox, and Edge.
